撰寫系統：macOS Sierra 10.12.5

使用語言：python 3.6.1

input file :
	glass_identification.csv
	hepatitis.csv
	image_segmentataion.csv
	pima_indians_diabetes.csv
	
執行方式：
	1.使用cmd執行：
		針對每個不同input file 在cmd輸入：
			python3 NB_glass.py
			python3 NB_hepatitis.py
			python3 NB_image_segmentataion.py
			python3 NB_pima_indians_diabetes.py
	2.使用exe執行：
		針對每個不同input file ，直接開啟該input file對應的exe檔即可。
